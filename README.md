# MAC Gyver's Maze



A simple 2D labyrinthe game where Mac Gyver had to escape. The aim of the game is to collect the 3 items in the maze and arrive on guardian's position in order to put him to sleep using these 3 items.




## Controls
 Please use **"S"** to start the game
 then use **keyboards arrows to move**
